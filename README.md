# bigscreen-remote

Web-based TV remote for KDE Plasma Bigscreen. Connects to a [remote-touchpad](https://github.com/Unrud/remote-touchpad) backend via WebSocket and provides a TV-optimized UI with d-pad navigation, media controls, and hold-to-repeat.

## Features

- D-pad with arrow keys, Enter/OK, Back/Escape
- Media controls (play/pause, volume, skip)
- Touchpad mode for mouse input
- Text input with mobile keyboard
- PWA support (installable on iPhone/Android home screen)
- No app install required — runs in any browser

## Usage

Open `https://jkhoeini.github.io/bigscreen-remote/` on your phone or computer, enter your remote-touchpad server address, and start controlling your TV.

## Requirements

- [remote-touchpad](https://github.com/Unrud/remote-touchpad) running on the target machine
- Any modern browser on your phone/tablet/computer
